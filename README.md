# ArvanScanner (آروان اسکنر) 🚀

آروان اسکنر یک نرم‌افزار دسکتاپ بومی و سریع (ساخته شده با Wails و Go) برای اسکن و پیدا کردن آی‌پی‌های سالم و تمیز (Clean IPs) از شبکه توزیع محتوای آروان‌کلاد (ArvanCloud) است. 

این برنامه با رابط کاربری جذاب و تاریک، امکانات پیشرفته‌ای برای تست لتنسی (Ping/Latency)، تست جیتتر (Jitter) و بررسی اتصالات TLS/TCP در اختیار شما قرار می‌دهد.

## ✨ ویژگی‌های کلیدی
- **رابط کاربری مدرن:** طراحی شده با رنگ‌بندی اختصاصی آروان.
- **به‌روزرسانی خودکار لیست آی‌پی‌ها:** دریافت خودکار آخرین رنج‌های آی‌پی مستقیماً از وبسایت آروان.
- **چندگانه (Concurrency):** امکان تنظیم تعداد کانکشن‌های همزمان (تا ۲۰۰ اسکن موازی) برای سرعت فوق‌العاده.
- **حالت‌های اسکن:** پشتیبانی از اسکن ساده TCP و همچنین اسکن پیشرفته TLS با امکان تنظیم نام دامنه (SNI).
- **مرتب‌سازی نتایج:** امکان سورت کردن نتایج بر اساس آی‌پی، لتنسی و جیتر.
- **خروجی سریع:** قابلیت کپی کردن تمامی آی‌پی‌های سالم با یک کلیک.

## 🛠️ پیش‌نیازها
برای اجرای کدها یا کامپایل برنامه، ابزارهای زیر نیاز هستند:
- [Go](https://golang.org/dl/) (نسخه ۱.۱۸ به بالا)
- [Node.js](https://nodejs.org/) (برای بخش فرانت‌اند)
- [Wails](https://wails.io/) (فریم‌ورک ساخت برنامه‌های دسکتاپ)

## 🚀 نحوه کامپایل و اجرای سورس کد
۱. ابتدا کدها را دانلود یا کلون کنید.
۲. وارد پوشه پروژه شوید:
```bash
cd ArvanScanner
```
۳. برای اجرای برنامه در حالت توسعه (Development):
```bash
wails dev
```
۴. برای بیلد گرفتن و ساخت فایل اجرایی نهایی (exe):
```bash
wails build -upx
```
فایل اجرایی نهایی در پوشه `build/bin/ArvanScanner.exe` قرار خواهد گرفت.

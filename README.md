# اسکنر آروان (ArvanScanner) 🚀

آروان اسکنر یک نرم‌افزار دسکتاپ بومی، سبک و بسیار سریع است که برای پیدا کردن آی‌پی‌های سالم و تمیز از شبکه توزیع محتوای آروان‌کلاد (ArvanCloud) طراحی شده است. 

این ابزار به شما کمک می‌کند تا با بررسی پینگ (Ping)، میزان تأخیر و کیفیت اتصال، بهترین و سریع‌ترین آی‌پی‌ها را پیدا کنید.

## ✨ امکانات برنامه
- **دریافت خودکار آی‌پی‌ها:** گرفتن جدیدترین لیست آی‌پی‌ها به صورت مستقیم از سایت آروان در کسری از ثانیه.
- **سرعت بالا (موازی‌سازی):** اسکن همزمان ده‌ها آی‌پی برای رسیدن به نتیجه در کوتاه‌ترین زمان ممکن.
- **بررسی دقیق پینگ و نوسان (Jitter):** نمایش دقیق میزان تاخیر و نوسان شبکه برای انتخاب پایدارترین اتصال.
- **تست پورت و امنیت (TLS/TCP):** امکان بررسی وضعیت پورت‌ها و ارسال درخواست‌های امن با نام دامنه دلخواه (SNI).
- **محیط کاربری ساده و زیبا:** کارکرد بسیار آسان و بدون نیاز به تنظیمات پیچیده.
- **مرتب‌سازی نتایج:** امکان مرتب کردن لیست خروجی بر اساس کمترین پینگ یا بهترین وضعیت.
- **کپی سریع:** ذخیره تمام آی‌پی‌های سالم در کلیپ‌بورد با یک کلیک.

## 🛠️ راهنمای اجرای سورس کد
اگر برنامه‌نویس هستید و می‌خواهید کدها را اجرا یا کامپایل کنید، به ابزارهای زیر نیاز دارید:
- [Go](https://golang.org/dl/) (نسخه ۱.۱۸ به بالا)
- [Node.js](https://nodejs.org/)
- [Wails](https://wails.io/)

**مراحل اجرا:**
۱. کدها را کلون کرده و در محیط خط فرمان وارد پوشه `ArvanDesktop` شوید:
```bash
cd ArvanDesktop
```
۲. برای اجرای برنامه در حالت توسعه:
```bash
wails dev
```
۳. برای ساختن فایل اجرایی (خروجی ویندوز):
```bash
wails build -upx
```
پس از پایان این مرحله، فایل اجرایی شما در مسیر `build/bin/ArvanScanner.exe` ساخته می‌شود.
